# Figma Clone

Прототип Figma, полурабочий проект, где можно "потыкаться" (разрабатывается в настоящий момент). 

![Скриншот](https://i.postimg.cc/7ZVzwDYP/figmaclone.png)

## Live Demo
[Посмотреть Демо](https://figma-clone-flax.vercel.app/)

## Возможности
- **Инструменты рисования** - выделение, перемещение, трансформация
- **Управление слоями** - система слоев
- **Экспорт в PNG, JPG, PDF, SVG** - сохраняйте свои работы

## Технологии
- **Frontend**: Next.js, React 18, TypeScript, Konva.js, Tailwind CSS
- **State Management**: React Hooks, Zustand
- **Деплой**: Vercel
